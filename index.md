# Rede Neural / Regressão Linear Simples

Este repositório contém um notebook de exemplo para treinar um modelo linear simples usando NumPy e visualização com Matplotlib.

## Arquivos

- `main.ipynb` - Notebook principal com:
  - geração de dataset linear com ruído
  - definição do modelo linear
  - cálculo de erro quadrático médio (MSE)
  - treinamento por descida de gradiente
  - visualização dos resultados

## Como usar

1. Abra o notebook `main.ipynb` no Jupyter ou no VS Code.
2. Execute as células na ordem.
3. Verifique os gráficos e os parâmetros aprendidos.

## Dependências

- Python 3
- NumPy
- Matplotlib

## Sugestão

Para rodar localmente, instale as dependências e abra o notebook:

```bash
python -m pip install numpy matplotlib
code main.ipynb
```
